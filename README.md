# Bsoft Offline

Bsoft Offline is a professional Flutter application designed for distribution and sales management in offline-first environments.

### Core Features
- **Offline Reliability**: Full operation without internet using local SQLite database.
- **Sales Management**: Workflow for recording sales and managing transactions.
- **Stock Control**: Real-time tracking of inventory and stock movements.
- **Client Ledger**: Manage client balances, details, and transaction history.
- **Direct Printing**: Integrated support for Bluetooth thermal printers.
- **Reporting**: PDF generation for invoices and data export (CSV/XML).

---

## Getting Started

This project is a Flutter application. Ensure you have the Flutter SDK installed and configured.

1.  **Clone the repository**: `git clone https://github.com/BacheneSoft/Bsoft_off.git`
2.  **Get dependencies**: `flutter pub get`
3.  **Run the app**: `flutter run`
